分享一下 hugo 的语法模板，如何获取总文章数和总字数，效果可以参考我的首页。

```html
{{ $articleCount := len .Site.RegularPages }}
{{ $totalWordCount := 0 }}
{{ range .Site.Pages }}
{{ $totalWordCount = add $totalWordCount .WordCount }}
{{ end }}
```

```html
<p>已经写了 {{ $articleCount }} 篇文章，共 {{ $totalWordCount }} 字。</p>
```
